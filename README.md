joopal_cam
===============================

version number: 0.0.1
author: Zach Fogg

Overview
--------

A selfie neural network classifier. Made at Los Altos Hacks by @zfogg and @kern.

Installation / Usage
--------------------

To install use pip:

    $ pip install joopal_cam


Or clone the repo:

    $ git clone https://github.com/zfogg/joopal_cam.git
    $ python setup.py install
